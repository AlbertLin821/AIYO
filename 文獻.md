# 旅遊推薦系統相關文獻

## 多模態旅遊推薦（圖片＋文字）

### SelfAM-Vtrans
多模態旅遊路線推薦，使用 Vision Transformer + LSTM + 自注意力做圖片與文字融合，提升個人化推薦效果。

參考文獻：https://pmc.ncbi.nlm.nih.gov/articles/PMC11628496/

### Multi-modal Travel Recommendation Algorithm with Multi-vector Word Sense Segmentation
用多向量語意切分與資料挖掘設計旅遊推薦演算法。

參考文獻：https://peerj.com/articles/cs-1436.pdf

## 時序與軌跡分析

### TMS-Net（Temporal Multilayer Sequential Neural Network）
用時序分割旅遊軌跡改善路線個人化效果。

參考文獻：https://www.nature.com/articles/s41598-024-84581-z

## 語意片段推薦

### Semantic Video Trailers（Google Research）
針對單支影片，把影片切成語意片段並依據使用者查詢排序，產生「語意濃縮的 Trailer」，概念與片段推薦很貼近。

參考文獻：https://static.googleusercontent.com/media/research.google.com/zh-TW//pubs/archive/45575.pdf

## 旅遊路線與距離計算

### Google Maps API、Distance Matrix API
用於計算旅遊路線與距離。

參考文獻：https://developers.google.com/youtube/v3/getting-started?hl=zh-tw

## LLM 旅遊規劃

### LLM Trip Planning: New Hybrid Method（Google 研究）
結合 LLM 創意與局部搜尋最佳化旅遊行程。

參考文獻：https://www.youtube.com/watch?v=veb_TwoxzMQ

### AI Travel Planner
LLM + Real-Time API 生成個人化行程，展示如何用 LLM 整合地圖與推薦。

參考文獻：https://www.youtube.com/watch?v=fI8ygOauNNA
