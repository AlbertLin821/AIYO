## 4. 收入來源與盈餘

### 4.1 商業模式：Freemium

假設商業化採用 Freemium 模式：

#### Premium 訂閱

- **價格**：每月 99 元
- **轉換率**：約 5% MAU 轉為付費
- **功能差異**：
  - 免費版：每日 10 次 AI 對話、基本行程規劃
  - Premium：無限對話、進階行程規劃、多人共編、匯出功能

#### 廣告收入

- **CPM**：80–120 元（台灣市場）
- **曝光量**：每月頁面曝光約 2.5M（10k MAU 估算）

#### Affiliate 合作

- **合作夥伴**：Klook / Booking.com / Agoda
- **抽佣比例**：約 5–10%
- **轉換率**：約 2–5% 使用者會透過連結訂購

### 4.2 收入試算（以 10k MAU 為例）

| 收入項目 | 金額 (TWD/月) | 計算方式 |
| :-- | :-- | :-- |
| Premium（99 × 500） | 約 49,500 | 10k × 5% × 99 |
| Ads（CPM100 × 2.5M/1000） | 約 250,000 | 僅為樂觀估，實際可能低 |
| Affiliate | 約 20,000–50,000 | 依實際轉換而定 |
| **總收** | **約 90k～300k+** | |

### 4.3 盈餘分析

#### 10k MAU 等級

| 項目 | 金額 (TWD/月) |
| :-- | :-- |
| **總收入** | 90k～300k+ |
| **總成本** | 約 52,000 |
| **盈餘** | 40k～250k+（依實際使用而變） |

#### 關鍵指標

- **LTV/CAC**：約 17（來自畢專報告）
- **回本期**：約 8 個月左右（若行銷做得好）

**備註**：

- 廣告收入為樂觀估算，實際可能較低
- Affiliate 收入需視實際轉換而定
- 初期可能為負盈餘，需有足夠資金支撐

## 5. 成本優化策略

### 5.1 開發階段

1. **使用免費/低價服務**：
   - GitHub Actions（CI/CD）
   - Vercel（前端部署，免費方案）
   - Supabase（資料庫，免費方案）

2. **本地開發優先**：
   - 使用 Ollama 本地開發，減少 API 成本
   - 使用 Docker Compose 本地測試

3. **開源工具優先**：
   - 使用開源監控工具（Prometheus + Grafana）
   - 使用開源 CI/CD 工具

### 5.2 營運階段

1. **GPU 資源優化**：
   - 使用 Spot Instance 降低 60–70% 成本
   - 使用量化模型降低記憶體需求
   - 實作模型快取與批次處理

2. **資料庫優化**：
   - 使用讀寫分離
   - 實作快取層（Redis）
   - 定期清理舊資料

3. **CDN 與快取**：
   - 使用 CDN 降低頻寬成本
   - 快取靜態資源與 API 回應

### 5.3 監控與自動化

1. **自動擴展**：
   - 根據流量自動調整 GPU 實例數量
   - 使用 HPA（Horizontal Pod Autoscaler）

2. **成本監控**：
   - 設定成本告警
   - 定期檢視資源使用情況
   - 優化不必要的服務

## 6. 風險評估

### 6.1 技術風險

- **模型效能**：本地模型可能不如 OpenAI API
  - **緩解**：使用 RAG 強化、優化 Prompt
- **擴展性**：GPU 資源可能不足
  - **緩解**：使用雲端 GPU 服務、實作負載平衡

### 6.2 成本風險

- **初期虧損**：使用者成長需要時間
  - **緩解**：準備足夠資金、分階段擴展
- **成本超支**：實際成本可能高於預估
  - **緩解**：定期檢視、設定預算上限

### 6.3 市場風險

- **競爭**：市場上已有類似服務
  - **緩解**：強化差異化功能、提升使用者體驗
- **使用者接受度**：可能不如預期
  - **緩解**：持續優化、收集使用者回饋

## 7. 不同階段的成本規劃

### 階段 1：MVP（0–1k MAU）

- **目標**：驗證產品概念
- **成本**：約 3,800 TWD/月
- **重點**：使用免費/低價服務，快速迭代

### 階段 2：成長期（1k–10k MAU）

- **目標**：擴大使用者基礎
- **成本**：約 13,000–52,000 TWD/月
- **重點**：優化成本結構，提升轉換率

### 階段 3：擴展期（10k–50k MAU）

- **目標**：規模化營運
- **成本**：約 52,000–110,000 TWD/月
- **重點**：自動化營運，提升效率

### 階段 4：成熟期（50k+ MAU）

- **目標**：穩定營運與獲利
- **成本**：約 110,000–220,000 TWD/月
- **重點**：持續優化，探索新收入來源
